[POI2014] PAN-Solar Panels

对于 nnn 组询问，每组询问给定四个整数 a,b,c,da,b,c,da,b,c,d，从区间 [a,b][a,b][a,b] 和 [c,d][c,d][c,d] 中任意选取两个整数 xxx 和 yyy，求 gcd⁡(x,y)\gcd(x,y)gcd(x,y) 的最大值是多少。

对于 100%100\%100% 的数据，1≤n≤10001 \le n \le 10001≤n≤1000，1≤a≤b≤1091 \le a \le b \le 10^91≤a≤b≤109，1≤c≤d≤109​1 \le c\le d \le 10^9​1≤c≤d≤109​。

[POI2015] ODW

给定一棵 nnn 个点的树，树上每条边的长度都为 111，第 iii 个点的权值为 aia_iai​。

[POI2000] 病毒

二进制病毒审查委员会最近发现了如下的规律：某些确定的二进制串是病毒的代码。如果某段代码中不存在任何一段病毒代码，那么我们就称这段代码是安全的。现在委员会已经找出了所有的病毒代码段，试问，是否存在一个无限长的安全的二进制代码。

例如如果 {011,11,00000}\{011, 11, 00000\}{011,11,00000} 为病毒代码段，那么一个可能的无限长安全代码就是 010101…010101 \ldots010101…。如果 {01,11,000000}\{01, 11, 000000\}{01,11,000000} 为病毒代码段，那么就不存在一个无限长的安全代码。

现在给出所有的病毒代码段，判断是否存在无限长的安全代码。

1≤n≤20001 \leq n \leq 20001≤n≤2000，所有病毒代码段的总长度不超过 3×1043 \times 10^43×104。

考虑建出 AC 自动机，记发生匹配的结点为危险结点，发现一段无限长的安全代码即为在 AC 自动机上的一条无限长的路径，且满足路径不经过危险结点，则直接在 AC 自动机上 dfs\rm dfsdfs 判环即可，时间复杂度 O(∑leni)O(\sum len_i)O(∑leni​)。

[POI2014] RAJ-Rally

给定一个 nnn 个点 mmm 条边的有向无环图，每条边长度都是 111。

  3. u​u​u​ 的拓扑序比 i​i​i​ 小，v​v​v​ 的拓扑序比 i​i​i​ 大。

[POI2004] JAS

它包含 n​n​n​ 个洞室和一些隧道连接他们。

每个洞室之间只有一条唯一的路径连接他们。Hansel 在其中一个洞室藏了宝藏，但是它不会说出它在哪。Gretel 知道，当她询问一个洞室是否有宝藏时，如果她猜对了 Hansel 会告诉她，如果猜错了他会告诉她哪个方向会有宝藏。 给出洞穴的信息，那么无论 Hansel 把宝藏藏在了哪，求出最少要询问多少次才能找到宝藏。

[POI2006] PAL-Palindromes

Johnny 喜欢玩文字游戏。

他写下了 n​n​n​ 个回文串，随后将这些串两两组合，合并成一个新串。容易看出，一共会有 n2​n^2​n2​ 个新串。

两个串组合时顺序是任意的，即 a 和 b 可以组合成 ab 和 ba，另外自己和自己组合也是允许的。

现在他想知道这些新串中有多少个回文串，你能帮帮他吗？

保证所有串只包含小写字母，且所有串两两不同，所有回文串的长度和不超过 2×1062 \times 10^62×106。

若不保证回文串，可以建出 Tire 树并在 Tire 树上预处理一些信息做到时间 O(n)O(n)O(n)，空间 O(n∣∑∣)O(n\left|\sum\right|)O(n∣∑∣)。

但题目保证了回文串，因此有更加优美的性质：两个回文串相加后仍是回文串，当且仅当这两个回文串的最小循环节相同。

因此只需 kmpkmpkmp 求出每个字符串的最小循环节并用哈希加 map 维护相等的串的对数即可。

时间复杂度 O(nlog⁡n)O(n\log n)O(nlogn)。
